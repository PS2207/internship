To create repository go to new on github & create a Repository with the same name as local folder name.

Now Open cmd inside the folder & run these commands:-
1)git init
2)git add .
3)git commit -m "first commit"
4)git branch -M main
5)git remote add origin https://github.com/PS2207/internship.giti.e(Repositoryname.git)
6)git push -u origin main
